I need to implement a "God-Tier" WebGL Video Player with a "Cloth Wave" animation effect, inspired by Lusion.co.

When the user clicks "Play Video", instead of a boring square appearing, I want a 3D Plane to float onto the screen, waving like a flag/cloth, and then stabilizing.

### 1. DEPENDENCIES
Install the 3D graphics stack:
`npm install three @types/three @react-three/fiber @react-three/drei`

### 2. THE SHADER LOGIC (The Secret Sauce)
We need a custom `ShaderMaterial` to distort the video mesh.
- **Vertex Shader:** Manipulate the `Z` position of vertices using a sine wave function based on `uTime` and `uv.x`.
  - Formula approximation: `pos.z += sin(uv.x * 10.0 + uTime) * uAmplitude;`
- **Fragment Shader:** Simply map the video texture to the mesh.

### 3. COMPONENT: `src/components/WavingVideo.tsx`
Create a React Three Fiber component.

**Requirements:**
- **Geometry:** `planeGeometry(args=[16, 9, 32, 32])` (High segment count is crucial for smooth waves).
- **Texture:** Use `useVideoTexture` from `@react-three/drei` to load the video file.
- **Animation Loop:**
  - On Enter: High wave amplitude (wild waving).
  - Transition: Use `framer-motion` or `useFrame` to damp the amplitude down to 0 (flat screen) over 2 seconds.
- **Interaction:**
  - Mouse Move: Subtle "tilt" effect based on mouse position.
  - Hover: The wave gently restarts (`amplitude = 0.2`).

### 4. THE OVERLAY CONTAINER
Create a wrapper component `src/components/VideoModal.tsx`.
- **Z-Index:** 100 (Top of everything).
- **Background:** `bg-black/90` backdrop blur.
- **Canvas:** The R3F Canvas should fill the center of the screen.

const vertexShader = `
  uniform float uTime;
  uniform float uAmplitude;
  varying vec2 vUv;
  void main() {
    vUv = uv;
    vec3 pos = position;
    // The "Flag" Wave Math
    pos.z += sin(pos.x * 2.0 + uTime * 2.0) * uAmplitude;
    pos.y += cos(pos.x * 2.0 + uTime * 2.0) * (uAmplitude * 0.5);
    gl_Position = projectionMatrix * modelViewMatrix * vec4(pos, 1.0);
  }
`;

const fragmentShader = `
  uniform sampler2D uTexture;
  varying vec2 vUv;
  void main() {
    gl_FragColor = texture2D(uTexture, vUv);
  }
`;
